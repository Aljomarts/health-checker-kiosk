# Health Self-Check Kiosk

A simple web application that lets a user enter their name, age, sex, weight, and height to compute their Body Mass Index (BMI), view a health category with a tailored recommendation, and have the submission recorded to a Google Sheet for staff to review.

# Problem It Solves

Many students and employees on campus have no quick, private way to check their BMI without visiting the clinic. This kiosk allows users to self-check their BMI and receive an instant, personalized wellness recommendation, while also giving the school nurse or HR office a record of usage so they can follow up with at-risk individuals.

# Features


Responsive form for collecting name, age, sex, weight, and height
Client-side validation of all required fields before processing
Automatic BMI calculation and classification into Underweight, Normal, Overweight, or Obese
Color-coded result card with a tailored health recommendation
Session history list showing recent submissions
Submission data sent to a Google Sheet via Google Apps Script
